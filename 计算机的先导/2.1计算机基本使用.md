## 计算机硬件相关
1. CPU：中央处理单元，简称CPU。是计算机的大脑，负责数据运算和处理。它长什么样子呢？目前生产计算机CPU的有两大厂商，一家是英特尔，一家是AMD。
	1. 核心数：核心数是一个CPU最重要的部分，目前的处理器普遍为多核处理器，核心越多普遍性能越好，并行化运算能力越强（核心数越多，CPU能同时处理的任务就越多，性能越强）。
	2. 线程数：比喻成马路，马路越多，处理器性能越强劲
	3. 频率：每秒完成多少指令周期（脉冲），也就是CPU执行速度
	4. 缓存：L3三级缓存是什么？
2. GPU：图形处理单元，简称 GPU。早期计算机是没有 GPU 的，因为那个时候没有图形相关的需求，只使用命令。后来有了GUI和3D图像等技术，显卡才逐渐出现在人们的视野
	1. 显存数：显存数类似于CPU与缓存的关系。不同的是，缓存和CPU很大一部分不是共享的，而显存在GPU中共享的。（你们可以参考苹果的M系列处理器的显存内存共享机制，这种机制使得苹果的笔记本在如今的AI时代大放异彩！）
	2. 核心数：核心数类比CPU的核心数，不同的是CPU的核心是相互独立存在的，而GPU是天然并行产物，所有核心同步计算！也就是说，在某一时刻，多个核心可以同时计算一个数据得到相同的结果。
3. RAM：计算机运行内存、或者直接叫内存，是计算机的核心部件之一。内存主要负责数据临时中转，将数据从硬盘转移到内存，在转移到CPU处理。
	1. DDR：代表了第几代内存技术，目前是 DDR 5
	2. 频率：表示内存每秒可传输数据的次数，单位为兆赫兹（MHz）或吉赫兹（GHz），它代表了内存的运行速度，频率越高，数据传输越快，整体系统性能理论上越好。
	3. 时序：衡量内存响应速度的一组数字参数，表示内存从接收指令到传输数据所需的时间。一般为CXX，比如C16、C18（在相同频率下，C16 内存的响应速度比C18 更快，因为它只需要16 个时钟周期来完成列地址访问，而 C18 需要 18 个。）。
4. HHD/SSD/M.2：硬盘/固态硬盘，是数据存储的另一个关键，它能够存储冷数据（不经常使用的数据），同时容量上十分大，你们手机有512g、256g实质上就是一种“硬盘”存储。
	1. HHD：机械硬盘，顾名思义是由机械部件构建的存储工具，其原理是磁存储，他的样子长这个样子。中间有多块磁盘，磁盘旁边是磁头，磁头负责偏转角度到磁盘上进行数据读取。他的速度不及SSD固态硬盘，但很大程度上我们今天还在使用它，因为其稳定可靠价格便宜
	2. SSD：固态硬盘，其内部是闪存颗粒，数据存储在其中，他的速度远高于机械硬盘（因为是电，而机械硬盘的速度取决于中间的电机，电机的旋转速度不可能快于电本身）。SSD其快速而被广泛适用于系统盘制作（你想让你的系统运行的更快？那就使用SSD），但是容量上不及HHD，且价格不算特别贵，几百块钱就能买到不错的SSD
	3. m.2固态：上述的两种硬盘都基于SATA总线进行的数据传输，SATA其本身就是一个大累赘...怎么办？我们想应该走一个更加快速的总线进行数据传输......PCIE总线，计算机当中最主干道！PCIE连接了计算机的几个专用设备，比如CPU、GPU、网卡、专业声卡、采集卡等等，在网络数据传输上和与CPU/GPU传输上直接连接、速度翻倍。m.2固态硬盘普遍较小，长这个样子，价格要比SSD更贵一点点。
	4. 关于这部分的拓展，你们可以看我之前在[米游社](https://www.miyoushe.com/ys/article/42638211)写的文章，或者来自b站[硬件茶谈](https://space.bilibili.com/14871346/search?keyword=%E7%A1%AC%E7%9B%98)的一系列硬盘科普视频

## 操作系统相关
### 操作系统有哪些？
1. windows：来自美国互联网巨头，当年福布斯第一人比尔盖茨的公司——微软研发的操作系统
	1. 家用：winXP、win7、win10、win11。win11 是你们现在常用的。
	2. 企业：win Server 系列。
2. Linux/Unix：来自林纳斯·托瓦丝，以及全世界开发者的智慧，凝结而成的操作系统。继承开源精神的伟大产物之一（开源，指将代码公开于世界，任何人可以任意修改使用，甚至商用）
	1. 因为Linux是开源系统，所以存在很多版本，比如最常见的三个：Ubuntu、Debin和已经死了的CentOS，它们都很好，都相似，都可以拿来做开发和使用。

作为开发者，我们需要熟悉Linux操作系统，这是必要的。

### GUI 图形化界面和 CLI 命令行界面
早期计算机是没有窗口的，最开始只有一个东西，叫终端（你们眼里黑客使用的东西，那个黑框框），或者叫tty（最早最早，叫法来自[电传打字机](https://en.wikipedia.org/wiki/Tty_(Unix))）。你们今天在windows上也可以见到类似的窗口，它的名字叫CMD，全称就是`Command命令`。你们可以按住`win+R`，输入`cmd`看到。

当然在`Linux操作系统中`，我们可以直接看到这个窗口，在这个窗口下，你同样可以完成在GUI下干的事情，只要你命令玩的 6。比如：你可以使用命令打开浏览器，可以打开音乐软件，可以操控任意的系统上的东西。这不是魔法，这是上古遗留下的“真经“。

CMD下的命令：
1. dir：查看当前目录文件
2. cd：进入某个文件夹
3. rd：删除文件/文件夹
4. ipconfig：查看当前网卡配置
5. ...

Linux下的命令：
6. ls：查看当前文件夹下的文件
7. clear：清空当前窗口
8. mkdir：创建文件夹
9. vim：创建文件
10. rm：删除文件/文件夹
11. ...

当然还有很多，我们目前没有必要学习那么多（事实上还需要再学一点），感兴趣你们可以参考这个链接进行学习：
